=== Spider View ===


Developer: Timo

Tags: visualisation, sensors

Requires: Chrome 32+


== Description ==


Spider View allows you to visualize all the present sensor values using D3.JS in your phone.


== How to use ==


1. Open Spider.html

2. Enter Device IP Address (eg. http://192.168.1.141:8765/). Click Connect

3. Make sure GPS settings are turned on

4. Drag the sensor from  available sensors to the active sensors. 

5. Click start.
