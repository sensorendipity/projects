=== DataSlicer ===

Developer: Chenchen

Tags: visualisation, sensor, recorder

Requires: Chrome 32+


== Description ==

DataSlicer gives you the graphical representation of how the sensor value changes in a defined time frame.


== How to use ==

1. Open index.html in Chrome

2. Enter Device IP Address (e.g. 192.168.1.141:8765). Please note, do not include “http://“. Click SET

3. Enter the record name and data sampling frequency (max 20 Hz)

4. Click Start and after sometime click Stop.

5. Graphs will display the value of Linear Acceleration, Gyroscope and Magnetic Field sensor.

6. You can also add sample name for a particular range.
